Простой шаг за шагом гид для запуска проекта

Кому это подходит
- Для начинающих и детей: тут объяснено просто, без сложных терминов.

Что нужно заранее
- Установлен Java (версия 11 или новее).
- Установлен Maven (чтобы собирать проект).
- Если используете IDE (IntelliJ IDEA, Eclipse), можно запускать прямо из неё.

Как запустить — самый простой вариант
1) Откройте проводник и перейдите в папку проекта:

   cd путь\до\папки\knowledge-app

2) Собрать проект (собирает все и упакует в папку target):

   mvn clean package

3) Запустить программу из IDE (проще всего для начинающих):
- Откройте проект в вашей IDE.
- Найдите класс с именем MainApp в пакете com.example.knowledge.
- Нажмите "Run" или запуск в IDE.

Если хотите запустить из собранного файла jar
- После `mvn clean package` откройте папку `target`.
- Там будет файл с расширением .jar. Запустите его так:

   java -jar имя-файла.jar

Советы и заметки
- Файл настроек базы данных: src/main/resources/db.properties. В нём указаны параметры для тестовой базы.
- По умолчанию в примерах может быть простой пароль для тестов. Для реальной работы поменяйте эти настройки.
- Если будут ошибки при запуске, скопируйте текст ошибки и пришлите мне — помогу разобраться.

Коротко и просто
- Сборка: mvn clean package
- Запуск в IDE: запустите класс MainApp
